# AGENTS Guide

This repository contains a Python + Playwright scraper for TripAdvisor China attractions.  
Primary script: `scrape_tripadvisor_attractions.py`.

## Goal

Automate extraction of attraction rank and name from TripAdvisor list pages, then output:

- Excel file (default: `attractionsForAgent.xlsx`)
- Screen recording (default: `attractionsForAgent.webm`)

The script supports configurable start page and page count.

## Repository Layout

- `scrape_tripadvisor_attractions.py`: main scraper and CLI entrypoint
- `requirements.txt`: Python dependencies
- `README.md`: user-facing usage notes
- `.tripadvisor_latest_outputs.json`: runtime state used to clean up previous outputs

## Environment Setup

Use Python 3 and install dependencies:

```bash
python3 -m pip install -r requirements.txt
python3 -m playwright install chromium
```

Optional for better video trimming: install `ffmpeg` (if unavailable, script still works and keeps full recording).

## Run Commands

Default run:

```bash
python3 scrape_tripadvisor_attractions.py
```

Example custom run:

```bash
python3 scrape_tripadvisor_attractions.py \
  --start-page 6 \
  --pages 5 \
  --output attractionsFromPage.xlsx \
  --video attractionsFromPage.webm
```

Useful flags:

- `--url` custom start URL
- `--start-page` page to begin from (1-based)
- `--pages` number of pages to scrape
- `--output` Excel output path
- `--video` recording output path
- `--headed` run with visible browser window

## Agent Editing Guidelines

When modifying scraper logic, preserve these behaviors unless explicitly requested otherwise:

1. **CLI compatibility**
   - Keep existing arguments and defaults stable.
2. **Data contract**
   - Excel sheet title remains `景点`.
   - Excel columns remain `编号`, `景点名称`.
3. **Pagination safety**
   - Navigation relies on detecting rank change after clicking next page.
   - Fail clearly if page content does not refresh.
4. **Output lifecycle**
   - Continue updating `.tripadvisor_latest_outputs.json`.
   - Keep cleanup of previous output files generated by earlier runs.
5. **Video handling**
   - Keep fallback path when `ffmpeg` is missing or trimming fails.

## Lightweight Validation

For non-network checks after code edits:

```bash
python3 -m py_compile scrape_tripadvisor_attractions.py
```

For functional checks (network/browser required), run with a small page count first:

```bash
python3 scrape_tripadvisor_attractions.py --pages 1 --start-page 1
```

## Notes for Future Agents

- The target website structure can change; selector breakages are expected maintenance.
- Keep waits and error handling explicit to avoid silent partial results.
- Prefer small, testable changes and verify output files are created as expected.
